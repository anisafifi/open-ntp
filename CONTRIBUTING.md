# Contributing to OpenNTP

Thanks for considering a contribution. This project welcomes issues, feature requests, and pull requests.

## Getting started

- Use Node.js 20+.
- Install dependencies: `pnpm install`.
- Run the dev server: `pnpm run dev`.

## Reporting issues

- Check existing issues first.
- Include reproduction steps, expected behavior, and actual behavior.
- Share logs or screenshots when helpful.

## Pull requests

- Keep changes focused and scoped.
- Update docs when behavior changes.
- Ensure formatting/linting passes: `pnpm run lint`.
- Provide testing notes and screenshots for UI changes.

## Code style

- Prefer clear, readable code over clever code.
- Keep functions small and well-named.
- Avoid unnecessary dependencies.

## Security

If you discover a security issue, please avoid filing a public issue. Email the maintainers with details.
